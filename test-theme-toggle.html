Theme Toggle Integration Test

This is a basic HTML file to open alongside the app for manual testing.

Test steps:

  1. Open http://localhost:3000 in your browser
  2. Look for the theme toggle icon in the top navigation bar (between search and user/cart icons)
  3. Click the theme toggle to open the menu
  4. Select Light, Dark, or System theme options
  5. Verify the app switches between themes properly
  6. Test responsive behavior by resizing window or using mobile view
  7. Check mobile menu drawer for theme toggle with labels
